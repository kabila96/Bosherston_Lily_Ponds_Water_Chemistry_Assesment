Bosherston Lily Ponds Water Chemistry Assesment

Prepared by Powell A. Ndlovu — Swansea University, Chevening Scholar 2025–2026

About this report

This repository includes a Streamlit application (app.py) that visualizes seasonal and long-term trends for Water Temperature, pH, and Conductivity across the Bosherston Lily Ponds and spring inflow, plus non-parametric statistics (Kruskal–Wallis), scatter plots with OLS regression, and correlation summaries (Pearson & Spearman).

How to run locally

pip install -r requirements.txt
streamlit run app.py
